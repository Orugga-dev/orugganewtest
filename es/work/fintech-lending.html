El desafío

Construir un negocio de crédito desde cero, en un entorno regulatorio complejo

El cliente necesitaba una plataforma completa de originación de créditos — no solo un frontend, sino todo el stack: una experiencia de solicitud de préstamo para el usuario, un backend cloud que pudiera manejar flujos de adquisición impulsados por campañas, e integraciones en tiempo real con bureaus de crédito y proveedores de identidad para evaluar solicitantes al instante.

La plataforma también necesitaba soportar múltiples campañas simultáneas con diferentes parámetros de préstamo, tasas y flujos de adquisición — configurables sin intervención del equipo técnico.

Qué construimos

Un stack de crédito completamente integrado

Frontend web responsive

Una interfaz de solicitud de préstamo rápida y mobile-first, diseñada para la conversión. Flujo de UX simple que guiaba a los solicitantes del interés a la presentación en minutos.

Backend cloud con flujos de campaña configurables

Un backend flexible que permitió al equipo del cliente configurar productos de préstamo, tasas y campañas de adquisición de forma independiente — sin cambios en el código para lanzar una nueva variante de producto.

Scoring crediticio en tiempo real vía múltiples APIs

Integraciones con múltiples bureaus de crédito y proveedores de verificación de identidad, llamados en tiempo real durante el flujo de solicitud para evaluar y aprobar solicitantes sin revisión manual para usuarios calificados.

Módulo de gestión de campañas

Un módulo de administración integrado para crear, gestionar y hacer seguimiento de campañas de adquisición de usuarios con parámetros personalizados — permitiendo al negocio iterar rápidamente sobre ofertas y canales.
